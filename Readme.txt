Перечисление дополнительных свойств с одинаковыми названиями для разных элементов:

RippleColor - цвет анимации Ripple effect

BackgroundMouseOver - цвет фона при наведении мыши

BorderBrushMouseOver - цвет обводки при наведении мыши

BackgroundChecked цвет фона при срабатывание события Checked

CornerRadius -  закругление краев у элементов



Реализовано в данный момент
ButtonRipple
RadioButtonRipple
RadioButtonRipple2
CheckBoxStyle_IOS
